# 更新日志

## v1.2.1 - Bug 修复

- **修复留言板提交失败**：前端 AJAX 请求（提交/回复/点赞）缺少 CSRF token，导致后端验证失败返回 403
- **修复 CSRF 失败响应格式不统一**：三个 API 端点返回 `error` 键而非 `success: false`，前端无法正确识别错误信息
- **修复留言板初始状态判断问题**：`guestbook.php` 现在自己在 `include header.php` 之前计算 `$guestbookEnabled`，避免依赖 `header.php` 的变量作用域导致的初始状态误判
- **统一安装向导的 JSON 写入方式**：`install.php` 现在使用与后台一致的 `writeJsonFile` 原子写入，替代原来的 `file_put_contents`

## v1.2 - 全面功能增强

### 新增功能（16项）

1. **Markdown 编辑器** - 后台日记编辑支持双模式切换：Summernote 富文本 与 Markdown 实时预览（marked.js）
2. **URL 重写** - 支持伪静态地址：`/diary/slug`、`/archive`、`/hot`、`/category/xxx`
3. **文章封面图** - 每篇日记可设置封面图片，前台列表卡片展示
4. **分类系统** - 后台完整 CRUD 管理分类（名称/别名/描述/排序），前台支持分类筛选与标签展示
5. **归档页面** - 按年/月分组展示所有已发布日记
6. **热门文章排行** - 基于浏览量的 Top 20 排行榜，前三名带奖杯标识
7. **LaTeX 数学公式** - 支持 `$...$` 行内公式与 `$$...$$` 块级公式渲染（KaTeX）
8. **代码块复制按钮** - 文章中的代码块右上角显示一键复制按钮
9. **社交分享** - 日记详情页支持复制链接、分享到微博/ Twitter
10. **文章浏览量计数** - 原子递增统计每篇文章浏览量
11. **邮件订阅系统** - 访客可订阅邮件通知，后台支持订阅者管理与批量发送
12. **评论邮件通知** - 新评论自动通知管理员，回复评论通知被回复者
13. **PWA 支持** - 添加 manifest.json 与 Service Worker，支持离线缓存访问
14. **图片处理流水线** - 上传图片自动生成缩略图（300px）、中等图（800px）与压缩原图
15. **后台数据图表** - 仪表盘集成 Chart.js，展示近 14 日 PV/UV 趋势与 Top 5 热门文章
16. **文章密码保护 + 定时发布** - 单篇日记可设密码访问，支持设定未来发布时间

### 优化与改进

- 完善 `.htaccess` URL 重写规则
- 优化图片上传返回格式，同时返回原图/缩略图/中等图地址
- 评论接口增加邮件通知钩子

## v1.1.1 - 安全修复

修复全面安全审计中发现的 44 项问题：

- **严重**：部分页面未鉴权、路径遍历、存储型 XSS、API 缺少 CSRF 防护
- **高危**：GET 请求执行状态变更、非原子写入、音乐管理安全问题
- **中危**：IP 掩码正则错误、评论图片上传绕过、安装重装缺少 CSRF
- **低危**：数据格式错误、PHP 版本过旧、HTML 双重编码
