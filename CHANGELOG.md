## [1.0] - 2026-07-12

### 新增

- **核心系统**：基于 PHP + JSON 的无数据库个人主页博客系统
- **日记系统**：富文本编辑（Summernote）、图片上传、标签分类、嵌套评论
- **日记增强**：草稿/发布状态、文章置顶、文章目录（TOC）、代码高亮（Prism.js）
- **留言板**：留言发布、嵌套回复、点赞系统、防重复点赞
- **标签云**：自动统计标签频率，按频率展示
- **全文搜索**：支持标题、内容、标签搜索，关键词高亮
- **RSS 订阅**：自动生成最近 20 篇日记的 RSS Feed
- **后台管理**：仪表盘、内容管理、日记管理、留言板管理、评论管理、音乐管理
- **设置中心**：UI 外观、主题色（6 种）、语言切换、功能开关
- **安全中心**：密码修改（8 位以上含字母和数字）
- **数据备份**：一键备份/恢复，自动排除敏感数据，含 Zip Slip 防护
- **操作日志**：审计日志记录、搜索、CSV 导出
- **访问统计**：PV/UV 统计 + 近 7 天图表展示
- **多语言**：中英文双语支持，后台一键切换
- **毛玻璃效果**：可调节模糊强度和透明度
- **深色模式**：自动检测系统偏好 + 手动切换
- **图片懒加载**：所有图片自动 loading="lazy"
- **图片点击放大**：Lightbox 遮罩层查看大图
- **阅读进度条**：日记详情页滚动进度指示
- **页面加载动画**：三点跳动加载器
- **评论邮件通知**：SMTP 邮件通知管理员
- **自定义错误页**：403 / 404 / 500 页面
- **安全测试页**：自动化安全功能验证

### 安全

- CSRF 防护：所有 POST 表单和 API 均有 Token 验证
- XSS 防护：HTML 消毒（白名单过滤）、CSP 头、输出转义
- 文件上传安全：finfo 真实类型检测、白名单扩展名、禁止 SVG、速率限制
- 认证安全：bcrypt 密码哈希、登录失败锁定（5 次/10 分钟）、Session 超时（1 小时）
- 安全 Cookie：HttpOnly、SameSite=Lax、Strict Mode
- JSON 原子写入：文件锁 + 临时文件 rename，防止并发数据损坏
- 速率限制：留言/回复/点赞/上传均有频率限制
- 目录保护：data/backups/uploads 禁止直接访问，uploads 禁止执行 PHP
- 安全响应头：CSP、X-Frame-Options、X-Content-Type-Options、Referrer-Policy
- 默认密码提示已移除，首次登录后强制修改
